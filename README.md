## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open your browser and navigate to `http://localhost:5173`

### Building for Production

To create a production build:

```bash
npm run build
```

The built files will be in the `dist` directory`. You can preview the production build with:

```bash
npm run preview
```

## Customization

### Updating Content

- **About Me**: Edit `src/components/About.tsx`
- **Skills**: Edit the `skills` array in `src/components/Skills.tsx`
- **Education**: Edit the `education` array in `src/components/Education.tsx`
- **Projects**: Edit the `projects` array in `src/components/Projects.tsx`

### Styling

The project uses Tailwind CSS. You can customize colors, spacing, and other design tokens in `tailwind.config.js`.
